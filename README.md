# cocos2d-x-particle-editor

This project allows you to create particle with cocos2dx. We have implemented all the properties and all properties can be changed in real-time to see the result. You can also export your particle as .plist. 

ImGui library was used in the project as GUI.
https://github.com/ocornut/imgui
